This repository contains multiple Machine Learning and Natural Language Processing (NLP) projects developed using Python, Scikit-learn, and Streamlit. 
These projects focus on text analysis, customer behavior prediction, and classification techniques.
**Projects Included**
1.**Food Sentiment Analysis :** 
A Natural Language Processing project that analyzes food reviews and predicts whether the sentiment is positive or negative using text classification techniques
2.**Spam Message Classifier:**
A Machine Learning model that classifies messages or emails as Spam or Ham (Not Spam) using NLP techniques.
3.**Customer Churn Prediction:**
A predictive analytics project that identifies customers likely to leave a service based on customer data and behavior patterns.
**Tools & Technologies:**
1.Python

3. Natural Language Processing (NLP)

**Deployment:**
These projects can be deployed using Streamlit Community Cloud.
 **Objective:**
 The objective of these projects is to apply Machine Learning and NLP techniques to solve real-world problems such as sentiment analysis, spam detection, and customer retention prediction.
